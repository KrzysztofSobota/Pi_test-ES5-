About the test

What is it?

"Pi test" is a simple and popular mathematical test which can be used in any programming language (like Java, Javascript, C, VisualBasic, etc.) having randomized function to generating numbers from a specified range.

How does it work?

The main idea is getting information about how big part of total generated points will be inside inscribed circle of the square.

No matter what is the length of the square edge ("a"), result of division inscribed circle area (S1) by square area (S2) is always the same and is equal 𝜋/4.

Because points must be evenly distributed, we can expect that during the test 𝜋/4 will be the permanent value (in theory) and because of that the calculated value multiply by 4 will give us "full" 𝜋 number.

Additional explanation

This is "𝜋 Test" based on ES5 Javascript (ES6 version you can get here).

Calculating method used here is very fast (2-3 sec for 10000 points) and will be perfect for person who need this test for something more than just some fun.

It is possible to connect test results with chart (for example to answer question "if sample size has an effect on 𝜋 value approximation?")
